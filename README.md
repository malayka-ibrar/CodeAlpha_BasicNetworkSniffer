# Basic Network Sniffer - CodeAlpha Internship Task 1

A Python-based network packet sniffer built with Scapy for educational purposes.

## Features
- Real-time packet capture and analysis
- IP, TCP, UDP protocol support
- Customizable BPF filters
- Detailed packet information display
- Capture summary report
- Cross-platform compatibility

pip install -r requirements.txt
